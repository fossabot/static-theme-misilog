## 💎<span id="GettingStarted"></span>Getting Started

Clone:

```
git clone https://github.com/misitebao/static-theme-misitebao.git
```

Installation dependencies:

```
npm install
```

Run locally:

```
npm run dev
```

Build:

```
npm run build
```

Local preview:

```
npm run serve
```
