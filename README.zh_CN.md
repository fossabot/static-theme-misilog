## 💎<span id="快速开始"></span>使用方法

下载仓库：

```
git clone https://github.com/misitebao/static-theme-misitebao.git
```

安装依赖：

```
npm install
```

本地运行：

```
npm run dev
```

打包构建：

```
npm run build
```

本地预览：

```
npm run serve
```
